# ATB-Pro-Curso-de-Terapia-Antimicrobiana
ATB-Pro** es una webapp académica interactiva desarrollada para la formación en **Infectología**, dirigida a médicos residentes de **Urgencias**, **Cuidado Crítico** y **Medicina Interna**.   Incluye contenido estructurado, tablas, figuras, audios, videos y un **quiz conectado con Google Forms** para evaluación académica.
